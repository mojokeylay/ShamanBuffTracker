# Shaman Buff Tracker
An addon that alerts the user when tracked shaman buffs are not active. This helps with the many weapon imbues or shields to track are not active, making sure you always have them enabled for every fight. Once the buff is on the player it simply is removed from the missing list, making it easy to track and determine whats missing.
